Steady State visual evoked potential MEG project. Full, left and right hemifield flickering checkerboard were presented to investigate the broadband signal.
Contents:
- Sessions with raw and processed data (**_SSMEG_'date')
- SSMEG specific code for preprocessing, data analysis and image plotting (Main function is runme_MEG_analysis_v2.m)
- An early version of Helena's denoise code (this can be deleted, because this code is moved to a git repository).
- Images to investigate several problems.
